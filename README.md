ATS Score Checker
Overview

ATS Score Checker is a full-stack AI-powered web application that helps job seekers optimize their resumes for Applicant Tracking Systems (ATS). Users can upload their resume, provide a job description, and receive an ATS compatibility score along with keyword analysis and AI-powered improvement suggestions.

Features
Resume Upload
Resume Text Extraction
ATS Compatibility Score
Keyword Matching
Missing Keywords Analysis
Resume Improvement Suggestions
